# odin-testing-practice

This project is part of [The Odin Project](https://www.theodinproject.com/) JavaScript curriculum.  
It contains practice implementations and unit tests for common JavaScript functions and classes, focusing on test-driven development.

## Features

- **Capitalize**: Capitalizes the first letter of a string.
- **Reverse String**: Reverses a given string.
- **Calculator**: Provides basic arithmetic operations (add, subtract, multiply, divide).
- **Caesar Cipher**: Implements the Caesar cipher with case and punctuation preservation.
- **Analyze Array**: Returns average, min, max, and length for an array of integers.

## Getting Started

### Prerequisites

- [pnpm](https://pnpm.io/) (recommended)
- Node.js (v18+ recommended)

### Install dependencies

```sh
pnpm install
```

### Run tests

```sh
pnpm test
```

Or to watch tests:

```sh
pnpm test:watch
```
