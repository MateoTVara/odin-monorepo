A classic Battleship game built as part of the [Odin Project](https://www.theodinproject.com/) JavaScript curriculum.

## Features

- Play against a computer AI with smart attack logic
- Random ship placement with option to reorder before starting
- Interactive UI with visual feedback for hits, misses, and sunk ships
- Responsive design for desktop and mobile devices

## Planned Features

- **Drag and drop ship placement:** Arrange your ships manually before starting the game
- **Two player mode:** Play against a friend on the same device

## Technologies Used

- **JavaScript (ES6+)**
- **HTML5**
- **CSS3** (custom styles, responsive design)
- **Webpack** (module bundling)
- **Jest** (unit testing)

## Getting Started

1. Clone the repository
2. Install dependencies with your package manager (e.g. `pnpm install`)
3. Run the development server:
   ```
   pnpm exec webpack serve
   ```
4. Open your browser to the provided local address

## Running Tests

```
pnpm test
```
